Panduan Command .setlistview & .resetlistview

Command .setlistview digunakan untuk mengatur tampilan custom pada hasil command .list. Command .resetlistview digunakan untuk mengembalikan tampilan list ke format default bot.

// Command Setlistview

.setlistview <template>

> Placeholder yang Tersedia

  • @judul – Judul tampilan
  • @tanggal – Tanggal
  • @waktu – Waktu
  • @listkey – Daftar produk
  • @namagroup – Nama grup
  • @deskgroup – Deskripsi grup
  • @owner – Owner bot
  • @user – Pengguna
  • @bot – Nama bot

// Help setlistview

> Contoh Penggunaan Setlistview (Markdown – Bisa Dicopy)

```text
.setlistview
LIST MENU GROUP @namagroup

Date : @tanggal
Time : @waktu

Pricelist Available:
• @listkey
```

// Setlistview berhasil

// Melihat Hasil Tampilan Baru

.list

// List custom

// Command Resetlistview

.resetlistview
[INFO]: Command ini akan menghapus template custom dan mengembalikan tampilan list ke format default.

// Alur Kerja Bot

  1. Template setlistview disimpan ke database grup
  2. Command .list menggunakan template tersebut
  3. Resetlistview menghapus template custom
  4. Tampilan list kembali ke default
