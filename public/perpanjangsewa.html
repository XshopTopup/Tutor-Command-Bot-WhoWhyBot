Panduan Penggunaan Command .perpanjangsewa

Command .perpanjangsewa digunakan untuk memperpanjang masa sewa bot di sebuah grup WhatsApp yang sudah aktif. Command ini hanya bisa digunakan di dalam grup tempat bot sedang disewa.

// Format Command Perpanjangsewa

.perpanjangsewa <durasi>

> Satuan Durasi

  • s = detik
  • m = menit
  • h = jam
  • d = hari
  • y = tahun

> Contoh Penggunaan

.perpanjangsewa 30d
.perpanjangsewa 1h
.perpanjangsewa 30m

// Contoh penggunaan command perpanjangsewa di grup

// Alur Kerja Bot Setelah Perpanjangsewa

  1. Bot memastikan command dijalankan di dalam grup
  2. Bot memvalidasi format durasi yang dimasukkan
  3. Bot mengambil data masa sewa grup saat ini
  4. Durasi baru ditambahkan ke sisa waktu sewa
  5. Bot menyimpan masa sewa terbaru ke database
  6. Bot mengirim notifikasi bahwa perpanjangan berhasil
[INFO]: Jika command dijalankan di chat pribadi, bot akan menolak dengan pesan bahwa command hanya bisa digunakan di grup.

// Bot mengonfirmasi perpanjang masa sewa

// Cek Sisa Sewa Setelah Perpanjangan

Setelah perpanjangan berhasil, gunakan command berikut untuk memastikan masa sewa telah bertambah: 

.ceksewa
[WARNING]: Pastikan melakukan perpanjangan sebelum masa sewa habis agar bot tidak otomatis keluar dari grup.
