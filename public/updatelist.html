Panduan Penggunaan Command .updatelist

Command .updatelist digunakan untuk mengubah daftar harga atau mengubah respon dari key produk yang sudah ditambahkan sebelumnya menggunakan command .addlist.

// Format Command Updatelist

.updatelist <key>@<deskripsi_baru>

> Keterangan Parameter

  • <key> : Nama produk yang ingin diperbarui.
  • <deskripsi_baru> : Respon atau daftar harga terbaru.
[WARNING]: Tidak boleh ada spasi sebelum atau sesudah simbol @.

> Contoh Format Salah

.updatelist abc @Tulis harga baru
.updatelist abc@ Tulis harga baru

// Bot menolak command karena format salah

> Contoh Format Benar

.updatelist abc@Tulis daftar harga baru kalian

// Bot berhasil memperbarui list produk

// Alur Kerja Bot Setelah Updatelist

  1. Bot memastikan command dijalankan di dalam grup
  2. Bot memvalidasi format key@deskripsi
  3. Bot mengecek apakah key sudah terdaftar
  4. Bot mengganti respon lama dengan respon baru
  5. Data diperbarui di database grup
  6. Bot mengirim notifikasi berhasil update
[INFO]: Jika key tidak ditemukan, bot akan menolak perintah dan meminta pengguna menambahkan list terlebih dahulu.

// Respon produk berubah sesuai data terbaru

// Melihat Hasil Update

Untuk memastikan data sudah berubah, gunakan: 

.list

Lalu ketik nama produk untuk melihat respon terbaru.
