Panduan Command !setproses, !changeproses, dan !delsetproses

Command pada halaman ini digunakan untuk mengatur, mengubah, dan mereset template pesan status PROSES pada sistem pesanan bot WHOWHYBOT. Template ini biasanya digunakan pada sistem order, jasa, atau transaksi.

// Command !setproses

Command !setproses digunakan untuk membuat dan menyimpan template pesan status PROSES. Template ini akan digunakan setiap kali pesanan berada dalam tahap diproses.

> Format Command

!setproses <text>

> Contoh Penggunaan

!setproses
Pesanan: @pesanan
Dari: @nama
Tanggal: @tanggal
Jam: @jam

> Placeholder yang Tersedia

  • @tanggal – Menampilkan tanggal saat ini
  • @jam – Menampilkan jam saat ini
  • @nama – Menandai pengguna yang dibalas
  • @pesanan – Menampilkan ulang pesan yang dibalas
[INFO]: Disarankan menggunakan command !setproses dengan cara reply ke pesan pesanan agar placeholder @nama dan @pesanan berfungsi dengan benar.

> Cara Kerja Bot

  1. Bot membaca teks template dari command
  2. Template PROSES disimpan ke database
  3. Placeholder diganti secara otomatis saat pesan dikirim
  4. Template digunakan setiap status pesanan PROSES

// Command !changeproses

Command !changeproses digunakan untuk mengubah template PROSES yang sudah dibuat sebelumnya tanpa perlu menghapus template lama.

> Format Command

!changeproses <text>

> Contoh Penggunaan

!changeproses
Pesanan: @pesanan
Dari: @nama
Tanggal: @tanggal
Jam: @jam

> Placeholder yang Tersedia

  • @tanggal – Menampilkan tanggal saat ini
  • @jam – Menampilkan jam saat ini
  • @nama – Menandai pengguna yang dibalas
  • @pesanan – Menampilkan ulang pesan yang dibalas
[WARNING]: Command !changeproses hanya dapat digunakan jika template PROSES sudah dibuat terlebih dahulu menggunakan command !setproses.

> Cara Kerja Bot

  1. Bot mengecek apakah template PROSES tersedia
  2. Jika tidak ada, bot menolak perintah
  3. Jika ada, template lama diganti dengan template baru
  4. Template PROSES baru langsung aktif

// Command !delsetproses

Command !delsetproses digunakan untuk menghapus template PROSES custom dan mengembalikan pesan PROSES ke format default bot.

> Format Command

!delsetproses

> Cara Kerja Bot

  1. Bot menghapus template PROSES dari database
  2. Bot mengembalikan format PROSES ke default
  3. Template custom tidak lagi digunakan
[INFO]: Setelah menggunakan !delsetproses, kamu dapat menggunakan !setproses kembali jika ingin membuat template baru.
