Panduan Command !setdone, !changedone, dan !delsetdone

Command pada halaman ini digunakan untuk mengatur, mengubah, dan mereset template pesan status DONE (pesanan selesai) pada sistem pesanan bot WHOWHYBOT.

// Command !setdone

Command !setdone digunakan untuk membuat dan menyimpan template pesan saat pesanan telah selesai (DONE). Template ini akan digunakan setiap kali status pesanan dinyatakan selesai.

> Format Command

!setdone <text>

> Contoh Penggunaan

!setdone
Pesanan: @pesanan
Dari: @nama
Tanggal: @tanggal
Jam: @jam

> Placeholder yang Tersedia

  • @tanggal – Menampilkan tanggal saat ini
  • @jam – Menampilkan jam saat ini
  • @nama – Menandai pengguna yang dibalas
  • @pesanan – Menampilkan ulang pesan yang dibalas
[INFO]: Gunakan command !setdone dengan cara reply ke pesan pesanan agar placeholder @nama dan @pesanan dapat berfungsi dengan benar.

> Cara Kerja Bot

  1. Bot membaca teks template DONE
  2. Template DONE disimpan ke database
  3. Placeholder diganti otomatis saat pesan dikirim
  4. Template digunakan setiap status pesanan selesai

// Command !changedone

Command !changedone digunakan untuk mengubah template DONE yang sudah dibuat sebelumnya tanpa harus menghapus template lama.

> Format Command

!changedone <text>

> Contoh Penggunaan

!changedone
Pesanan: @pesanan
Dari: @nama
Tanggal: @tanggal
Jam: @jam

> Placeholder yang Tersedia

  • @tanggal – Menampilkan tanggal saat ini
  • @jam – Menampilkan jam saat ini
  • @nama – Menandai pengguna yang dibalas
  • @pesanan – Menampilkan ulang pesan yang dibalas
[WARNING]: Command !changedone hanya dapat digunakan jika template DONE sudah dibuat terlebih dahulu menggunakan command !setdone.

> Cara Kerja Bot

  1. Bot mengecek apakah template DONE tersedia
  2. Jika belum ada, bot menolak perintah
  3. Jika ada, template DONE lama diganti
  4. Template DONE baru langsung aktif

// Command !delsetdone

Command !delsetdone digunakan untuk menghapus template DONE custom dan mengembalikan pesan DONE ke format default bot.

> Format Command

!delsetdone

> Cara Kerja Bot

  1. Bot menghapus template DONE dari database
  2. Format DONE dikembalikan ke default bot
  3. Template custom tidak lagi digunakan
[INFO]: Setelah reset, kamu dapat menggunakan !setdone kembali untuk membuat template DONE baru.
