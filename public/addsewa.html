Panduan Penggunaan Command .addsewa

Command .addsewa digunakan untuk menyewakan bot WhatsApp ke sebuah grup dalam jangka waktu tertentu. Setelah bot berhasil disewa, bot akan otomatis masuk ke grup dan aktif hingga masa sewa berakhir.

// Format Command Addsewa

.addsewa <link_grup> <durasi>

> Satuan Durasi

  • s = detik
  • m = menit
  • h = jam
  • d = hari
  • y = tahun

> Contoh

.addsewa https://chat.whatsapp.com/xxxxxx 5m

// Command addsewa

// Alur Kerja Bot Setelah Addsewa

  1. Bot memvalidasi link dan durasi
  2. Data sewa disimpan ke database
  3. Bot otomatis join grup
  4. Bot mengirim pesan aktivasi
  5. Timer sewa mulai berjalan

// Bot aktif di grup

// Cek Sisa Sewa

.ceksewa
[WARNING]: Bot akan otomatis keluar dari grup jika masa sewa habis dan tidak diperpanjang.
