Panduan Penggunaan Command .addlist

Command .addlist digunakan untuk menginjeksi daftar produk atau menu ke dalam database grup. List ini nantinya akan dirender saat command .list dieksekusi.

// Format Command Addlist

.addlist <key>@<deskripsi>

> Parameter Config

  • <key> : String nama produk (tanpa spasi).
  • <deskripsi> : String deskripsi produk atau value harga.
[WARNING]: Syntax Error akan terjadi jika ada spasi (whitespace) sebelum atau sesudah simbol @.

> Contoh Syntax Error (Salah)

.addlist abc @Tulis daftar harga
.addlist abc@ Tulis daftar harga

// Error Log: Format addlist invalid

> Contoh Syntax Valid (Benar)

.addlist abc@Tulis daftar harga kalian

// Success: Data berhasil disimpan ke database

// Runtime Alur Bot

  1. System mengecek command di dalam grup scope.
  2. System memvalidasi format string key@deskripsi.
  3. System parsing spasi di sekitar delimiter @.
  4. System menyimpan object list ke database.
  5. System mengembalikan callback sukses.

// Output: Daftar produk muncul saat command .list dipanggil

// Fetch Data List

.list

Bot akan me-return array produk yang telah ditambahkan. Untuk melihat detail object, ketik nama key di chat stream.
